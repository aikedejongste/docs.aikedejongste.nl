# Proton VPN

## Install the VPN

`curl https://repo.protonvpn.com/debian/dists/stable/main/binary-all/protonvpn-stable-release_1.0.3_all.deb`
`sudo apt install *.deb`
`sudo apt update && sudo apt install protonvpn-cli`
