# Ruby on Rails Dependencies

## MySQL database

```bash
apt install libmysqlclient-dev mysql-server-x.x
```

## Postgres database

```bash
bundle add pg
```

```bash
apt-get install libpq-dev
```

## Gem: ruby-filemagic

```bash
apt install libmagic-dev
```
